# Simple Counter Application

A beautiful, interactive counter application built with HTML, CSS, and JavaScript.

## Features

- **Increment/Decrement**: Use the "+" and "–" buttons to increase or decrease the counter
- **Reset Function**: Reset the counter back to 0 with the reset button
- **Keyboard Support**: 
  - `+` or `=` keys to increment
  - `-` key to decrement
  - `r` or `R` to reset
  - Arrow Up/Down keys for increment/decrement
- **Visual Feedback**: 
  - Counter changes color based on value (green for positive, red for negative)
  - Smooth animations and hover effects
  - Button click animations
- **Responsive Design**: Works on both desktop and mobile devices
- **Fun Features**:
  - Click counter that shows celebration after every 10 clicks
  - Hidden Konami code easter egg (↑↑↓↓←→←→)

## How to Use

1. Open `index.html` in your web browser
2. Click the "+" button to increase the counter
3. Click the "–" button to decrease the counter
4. Click "Reset" to set the counter back to 0
5. Use keyboard shortcuts for quick interaction
